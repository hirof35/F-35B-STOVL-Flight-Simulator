#F-35B STOVL Flight SimulatorTypeScriptとThree.jsで開発された、F-35B ライトニング II の飛行および短距離離陸・垂直着陸（STOVL）を再現した軽量な3Dフライトシミュレーターです。

通常の高速飛行モード（CTOL）から、3軸推力偏向ノズルとリフトファンを駆動させた垂直着陸モード（STOVL）へのダイナミックなモード遷移（コンバージョン）の物理演算とビジュアルをWebブラウザ上で体験できます。
🚀 特徴機能3軸推力偏向ノズルの連動 (3BSD)Spaceキーの入力と連動し、機体後部のノズルが3セグメントの階層構造でカチカチとリアルタイムに90度下向きへと折れ曲がります。
スロットルに連動したアフターバーナーエフェクトもノズルの向きに追従します。
F-35B特有の推力配分（リフトファン）ノズル角度が30度を超えると、コックピット後方の「リフトファン」が作動。
総推力の最大40%が前方に配分され、純粋な垂直推力でのホバリングを可能にします。
空気力学とストール（失速）シミュレーション通常モード時は前進速度に応じた揚力（Lift）と抗力（Drag）が発生します。
速度が 時速150km以下 に低下するか、機首を上げすぎて 迎角（AoA）が18度 を超えると、主翼の揚力が一気に15%まで激減する「ストール（失速）状態」に陥り、機体が自由落下を始めます。
地面効果（グラウンドエフェクト）の再現垂直着陸モード時、地面（高度12m以下）に接近すると、噴流が地面に跳ね返ることで最大20%の上向きの推力クッション（正の地面効果）が発生。
着陸寸前の独特な「粘り」を操縦感覚として再現しています。
🕹 操作方法 (Controls)キーボードを使って機体をコントロールします。
キー操作内容物理的な挙動W / Sピッチ (Pitch)機首上げ / 機首下げA / Dロール (Roll)機体を左右に傾けるQ / Eヨー (Yaw)機体を左右に水平旋回させるShiftスロットルUP推力を増加させる（アフターバーナー点火）CtrlスロットルDOWN推力を減少させるSpace (長押し)STOVL モードノズルを90度真下に向け、ホバーモードへ移行
💡 ヒント（ストールからの脱出方法）:低速で飛行中に失速（STALL WARNING: CRITICAL）になった場合、**「Shiftキーで出力を上げて機首を下げ、速度を稼ぐ」か、あるいはF-35B特有の機能として「Spaceキーを長押ししてノズルを真下に向け、推力で強引に浮き上がる」**ことで墜落を回避できます。
🛠 導入手順 (Setup)前提条件Node.js (v18以上推奨) がシステムにインストールされていること。1. 依存関係のインストールプロジェクトのルートディレクトリで以下のコマンドを実行し、必要なパッケージ（Vite, TypeScript, Three.js）をインストールします。Bashnpm install
2. 開発サーバーの起動ローカルの開発サーバーを起動してブラウザでシミュレーターを実行します。
Bashnpm run dev
起動後、ターミナルに表示されるURL（例: http://localhost:5173/）にブラウザでアクセスしてください。3. プロダクションビルド（配布用ファイルの生成）最適化された成果物を dist フォルダに出力します。Bashnpm run build
